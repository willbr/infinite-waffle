# infinite-waffle
A silly text editor, inspired by [OneNote](https://en.wikipedia.org/wiki/Microsoft_OneNote) and [Obsidian](https://obsidian.md/).

# keys

| key          | action          |
|--------------|-----------------|
| tab          | display toolbox |
| double click | create cell     |
